# Bootstrap Landing Page

This project is a responsive landing page built using Bootstrap 5.3.3. It features a navigation bar, a hero section with a video, and a call-to-action button.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling the webpage.
- **Bootstrap 5.3.3**: For responsive design and pre-built components.
- **Font Awesome**: For icons.
- **JavaScript**: For interactivity (e.g., tooltips).

## Features

- **Responsive Navbar**: A collapsible navigation bar that adapts to different screen sizes.
- **Hero Section**: Includes a heading, description, and a video.
- **Call-to-Action Button**: A button with a tooltip for additional information.


## Usage

- **Navbar**: Navigate through different sections of the page.
- **Hero Section**: Watch the video and read the description.
- **Call-to-Action Button**: Click the button to contact.

## Customization

- **Styles**: Modify `styles.css` to change the appearance.
- **Content**: Update the HTML to change text, images, and videos.

## Dependencies

- [Bootstrap 5.3.3](https://getbootstrap.com/)
- [Font Awesome 6.6.0](https://fontawesome.com/)
